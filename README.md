# stoic-quotes

Daily Stoic philosophy quote Discord bot. Fetches quotes from Dailystoic.com and Orion Philosophy, delivers one random Stoic quote each morning with author attribution and a practical takeaway.

## Topics
python, discord, bot, stoicism, philosophy, quotes

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/stoic-quotes.git
cd stoic-quotes

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 stoic-quotes.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
